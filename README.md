# r-and-d
A basic dice-roller built in Rust with minimal dependencies.
-
R&D is a (currently) CLI-only dice-roller built mainly for Dungeons & Dragons. (Pretend there's more text here; that's kind of all there is to it.)
Currently there is no build for Mac (either Intel or Silicon) because it's kind of a pain, and also I don't have an XCode SDK, but I am currently working on it.
